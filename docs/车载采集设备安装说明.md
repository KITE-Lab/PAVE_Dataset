# 车载采集设备安装说明

**版本**：v1.0  
**日期**：2025-09-22  
**作者**：李翔宇  
**单位/团队**：KITE Lab  

本文档用于指导车载采集设备的正确安装、接线与调试，确保数据采集过程的稳定性和安全性。请在操作前仔细阅读本说明。

---

## 1. 安装前准备

- 确认车辆处于 **熄火状态**，并拉紧手刹。
- 确认所有设备、线材和工具齐全。
- 建议两人配合进行安装：一人负责设备固定，另一人负责供电与布线。

## 3. 安装步骤

### 3.1 主机安装
1. 将工控机放置于 **中央扶手箱** 或 **副驾驶手套箱** 中，确保固定牢靠，避免行车中震动。  
2. 使用固定架或魔术贴进行防滑处理。  
3. 工控机设计为 **上电即自动启动并开始录制**，无需手动开关机；当车辆断电时，工控机会自动关机。  

### 3.2 摄像头安装
1. **前向摄像头**：2 个，安装在前挡风玻璃外侧中央偏左右位置，保证无遮挡。  
2. **左右侧摄像头**：分别安装于车辆外部左右两侧（靠近车门或车窗位置），保证水平视野无遮挡。  
3. **车内摄像头**：安装在合适位置，保证能完整拍摄 **中控屏幕**，避免反光与遮挡。  
4. 所有摄像头固定后，检查画面无遮挡，并连接至主机 USB 口。  

### 3.3 GNSS + IMU 安装
1. GNSS 天线固定在车顶中央，**纵向安装**（保证正北/车头方向一致），远离干扰源。  
2. IMU 固定在 **后备箱后轴中间位置**，方向与车身保持一致。  
3. GNSS 与 IMU 通过数据线连接至工控机。  

### 3.4 电源安装
1. 使用车载电源适配器，将车辆电源转换为主机/摄像头/传感器所需电压。  
2. 建议单独走线，避免与车内其他电器线路混杂。  
3. 接线完成后，使用万用表检测电压是否稳定。  

### 3.5 存储与调试
1. 插入 SSD/硬盘，确认存储设备已挂载成功。  
2. 启动车辆后，工控机会自动上电并启动录制程序，摄像头/GNSS/IMU 数据将自动采集。  
3. 断电时工控机会自动关机，数据会安全保存。  

---

## 4. 注意事项

- **安全第一**：安装过程中确保车辆断电，避免短路或触电。  
- **固定可靠**：所有设备需使用支架或固定带，避免因颠簸松动。  
- **自动运行**：工控机上电即启动录制，无需人工干预；车辆断电后自动关机。  
- **供电稳定**：避免直接从点烟器取电超过额定电流，建议使用专用电源模块。  
- **线缆管理**：所有线缆需走边缘、捆扎固定，避免影响驾驶安全。  
- **防尘防水**：车外摄像头和 GNSS 天线需做好防水处理。  
- **定期检查**：每次采集前检查供电、电缆连接和存储空间。  

---

## 5. 常见问题

1. **摄像头未识别**：检查 USB 口是否供电，是否超过带宽限制；必要时增加 USB Hub。  
2. **GNSS 无信号**：检查天线位置是否无遮挡，是否位于金属遮挡下方。  
3. **主机未启动**：确认电源模块是否正常供电，车辆点火后工控机应自动启动。  
4. **存储空间不足**：及时导出数据，并定期清理。
